# МЕГА ИГРА (2001)

![Обложка](cover.png)

**Простой шуточный сайт выдуманного игрового журнала «МЕГА ИГРА» №10 (89), октябрь 2001 года.**

Сайт полностью статический, сделан в стилистике начала 2000-х (таблицы, яркие акценты, старые веб-приёмы) и идеально подходит для публикации на **GitHub Pages**.

## Что внутри

- Точная копия настоящей обложки журнала (использована `cover.png`)
- 7 «статей» на основе обложки (Gothic, GTA III, Silent Hill 2, Max Payne, Шрек, Властелин Колец, интервью Linkin Park)
- Читы и коды 2001 года
- Топ-10 RPG осени
- Конкурс «Выиграй PS2»
- Гостевая книга (работает!)
- Письма читателей
- Много ретро-рекламы (GeForce 2, модемы 56k, CD с 1000 играми)
- Пасхалки (включая классический Konami Code)
- Полностью на русском языке

## Как запустить локально

Просто откройте `index.html` в любом браузере.

Или запустите локальный сервер:

```bash
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

## Как выложить на GitHub Pages

1. Создайте новый репозиторий на GitHub (можно назвать `mega-igra-2001` или любое другое имя).

2. Загрузите файлы в репозиторий:
   ```bash
   git init
   git add .
   git commit -m "Первый номер МЕГА ИГРА — октябрь 2001"
   git branch -M main
   git remote add origin https://github.com/ВАШ_НИК/ИМЯ_РЕПО.git
   git push -u origin main
   ```

3. В настройках репозитория (Settings → Pages) выберите:
   - Source: **Deploy from a branch**
   - Branch: `main`
   - Folder: `/ (root)`

4. Через 1–2 минуты сайт будет доступен по адресу:
   `https://ВАШ_НИК.github.io/ИМЯ_РЕПО/`

Готово! Теперь все могут оценить, какой была «МЕГА ИГРА» в 2001-м.
